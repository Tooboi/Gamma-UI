Container

Containers are the building block of Gamma UI

They are responsive flex boxes that snap to breakpoints

Must use both container and container-{size} for responsive behavior

When used with a row you are set to add anything to your page

This paragraph is inside a container, the code below is inside a row

<div class="container container-md">
  <div class="row">
    ...content...
  </div>
</div>

Use the naming format: container-{size}

container will be 100% width under the set breakpoint

Otherwise it will snap to each breakpoint

When using with a navbar you should add a mt-5 class to compensate for the navbar height
